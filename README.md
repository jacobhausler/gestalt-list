# Node Boilerplate

## Development

From a blank directory:

```
git clone https://github.phytel.com/tmartinez/node-boilerplate.git .
npm install
npm run dev
```

## Testing

Tests live alongside their sources as `test.js` files.

#### Single run

```
npm run test
```

#### Watch

```
npm run test:watch
```

## Deployment

```
npm run deploy
```
